## :computer: Usage

1. Clone the repository: `git clone https://github.com/Mohanbirajdar/HMS.git`
2. Navigate to the project directory: `cd HMS`
3. Compile the program: `gcc hms.c -o hms`
4. Run the program: `./hms`
